# 🔗 RECURSOS Y LINKS — QA JUNIOR

## APIs para practicar (gratis, sin registro)
- https://reqres.in — API REST simulada, ideal para Postman
- https://jsonplaceholder.typicode.com — Posts, comentarios, usuarios falsos
- https://dummyjson.com — Más completa que jsonplaceholder
- https://petstore.swagger.io — Ejemplo clásico de Swagger

## Apps para practicar testing manual y automatización
- https://saucedemo.com — App e-commerce para automatizar (usuario: standard_user / secret_sauce)
- https://the-internet.herokuapp.com — Bugs y casos especiales (alerts, iframes, auth)
- https://automationexercise.com — E-commerce completo con API incluida
- https://demoqa.com — Formularios, widgets, interacciones

## SQL
